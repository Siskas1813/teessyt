# Anime Tetris

Простое начало реализации игры "аниме Тетрис" для мобильных устройств. Игра написана на JavaScript и запускается в мобильном браузере. Код использует canvas для отображения фигур. Вместо классических блоков можно подставить спрайты аниме-девочек.

## Запуск

Откройте `index.html` на мобильном устройстве или в браузере. Экран автоматически масштабируется под размер устройства.

## Дальнейшее развитие

- Замените квадратные блоки на изображения из каталога `sprites`.
- Добавьте звуковые эффекты и музыку.
- Реализуйте сенсорное управление для мобильных устройств.
- Улучшите дизайн интерфейса в стиле аниме.
