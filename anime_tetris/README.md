# Rune Tac-Toe: Duel of Sigils (mobile prototype)

Браузерный мобильный прототип тактических «крестиков-ноликов» по концепту JRPG-дуэли:

- поле **5×5**;
- цель — собрать **4 в ряд**;
- есть ресурсы **мана/энергия**;
- есть способности: щит, заморозка, поджог клетки, рывок, убийство, ульта;
- есть особые клетки: мана, святилище, портал, проклятая.

## Запуск

Откройте `index.html` в мобильном браузере.

## Что уже реализовано

- пошаговый матч X vs O;
- выбор действия через панель кнопок;
- базовые статусы (щит, заморозка, проклятие, блок клетки);
- проверка победы и лог событий.
